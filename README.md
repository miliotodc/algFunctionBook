# algFunctionBook
These notebooks are companion material for the upcomming two-volume set "Algebraic Functions, A Computational Introduction for Undergraduated Using Mathematica, Volumes 1 and 2".  However, users are encouraged to refer to the volumes, when they are available, for a description of the topics and instructions for using the notebooks.  In particular several notebooks rely on data collected by other notebooks.  Understanding this relation will avoid unnecessary problems using the notebooks.  Additionally, the volume chapters go into great detail in how to use the notebooks with examples. Using and understanding the examples in the chapter tutorials will greatly improve success in using them as much of the code is sensitive to various parameters set for optimal performace and are explained in the tutorials.

I will update this file when the volumes become available.
